Description

Drop n load is a chunked uploader with drag and drop capabilities.

You can upload any number of files no matter its weight –tested up to 3GB– and the uploader splits them into chunks and starts uploading.

Configuration

You can configure it from index.html.

// Initialize Upload object
window.upload = new Upload({
    uploadScript: 'serverScript.php',
    uploadFolder: 'uploads/',
    tempFolder: 'tmp/',
    chunkSize: 200000,
    maxUploadSize: 2000000000,
    files: []
});
 
// Initialize Drag and Drop object
window.dragAndDrop = new DragAndDrop({
    droppableContainerId: 'droppable-container',
    borderStyleOnDragover: '6px dashed #444',
    uploadButtonId: 'upload-button',
    droppablePlaceholderId: 'droppable-placeholder'
});
 
function fire() {
    upload.uploadFile(0);
}


 Install

To install:

# In the document root of the webserver
git clone https://github.com/jlgarhdez/drop-n-load.git
cd drop-n-load
coffee --compile --output js/ coffee/
